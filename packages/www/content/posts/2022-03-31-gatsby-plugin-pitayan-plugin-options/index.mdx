# Use our plugin this way.

```js
// Just an example
{
  resolve: `@pitayan/gatsby-theme-pitayan`,
  options: {
    siteAssets: "src/assets",
    postsPerPage: 6,
    mailChimpEndpoint:
      "https://pitayanblog.us14.list-manage.com/subscribe/post?u=234bf6777b76872feb7d92a68&amp;id=27fad95f3b",
    mailChimpTimeout: 3500
  }
}
```

# The plugin options

| option                  | description                                                                                     | default             |
|-------------------------|-------------------------------------------------------------------------------------------------|---------------------|
| siteAssets              | The path of the static assets                                                                   | `js•"./src/assets"` |
| postsPerPage            | How many posts to be displayed in each list page                                                | `js•10`             |
| mailChimpEndpoint       | The embeded form endpoint for your MailChimp account                                            | `js•undefined`      |
| mailChimpTimeout        | The timeout (ms) for the MailChimp subscription API request                                     | `js•3500`           |
| gatsbyRemarkPluginsHead | The plugins for gatsby-remark-plugins that will be applied before @pitayan/gatsby-theme-pitayan | `js•[]`             |
| gatsbyRemarkPluginsTail | The plugins for gatsby-remark-plugins that will be applied after @pitayan/gatsby-theme-pitayan  | `js•[]`             |
